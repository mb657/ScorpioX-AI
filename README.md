# ScorpioX-AI – AI Browser Dashboard

**ScorpioX-AI** is a modern, frontend-only web app that allows users to **browse, explore, and use multiple AI tools** from a single interface. Think of it as a mini “AI Chrome” for AI productivity.

## Features

- **AI Tool Browser** – View tens of AI tools in a clean grid or list format  
- **Categories & Filters** – Browse by category: Text, Image, Code, Chat, Productivity, Research, etc.  
- **Tool Previews & Use** – Click a tool to interact via modal or embedded frame  
- **Favorites / Quick Access** – Save frequently used tools for faster access  
- **Responsive Design** – Works perfectly on desktop and mobile devices  

## Tech Stack

- **Frontend:** React + Tailwind CSS  
- **Hosting:** GitHub Pages / Vercel / Render  
- **APIs:** OpenAI or other public AI tool APIs
